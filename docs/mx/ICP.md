# ICP — Ideal Customer Profile (MX-First)

## Perfil del Despacho (B2B SaaS)

### Despacho Primario: Divorcio Incausado

| Atributo | Descripción |
|---|---|
| **Tamaño** | 1-10 abogados, despacho familiar o boutique |
| **Ubicación** | CDMX, Jalisco, Nuevo León, Estado de México |
| **Vertical principal** | Derecho familiar — divorcio incausado |
| **Volumen mensual** | 5-30 casos nuevos por mes |
| **Pain points** | Coordinación manual de documentos, seguimiento por WhatsApp sin estructura, expedientes incompletos, tiempo perdido en recopilación |
| **Tecnología actual** | WhatsApp personal, Excel/Google Sheets, correo electrónico |
| **Disposición a pagar** | $2,000 - $8,000 MXN/mes |
| **Decisor** | Abogado titular o socio fundador |

### Despacho Secundario: Consumidor / Laboral

| Atributo | Descripción |
|---|---|
| **Tamaño** | 1-5 abogados |
| **Vertical** | Quejas PROFECO/CONDUSEF, despido injustificado |
| **Volumen** | 10-50 consultas por mes |
| **Pain points** | Clasificación manual de casos, cálculo de prestaciones, seguimiento a clientes |

## Perfil del Cliente Final (B2C Prep Kit)

### Cliente Primario: Divorcio

| Atributo | Descripción |
|---|---|
| **Edad** | 28-50 años |
| **Género** | Predominantemente mujeres (65%) |
| **Ubicación** | Zonas urbanas MX (CDMX, GDL, MTY, Puebla) |
| **Situación** | Matrimonio con hijos menores, separación reciente o en proceso |
| **Canal preferido** | WhatsApp > búsqueda Google > redes sociales |
| **Dispositivo** | Móvil (85%+) |
| **Idioma** | Español MX |
| **Urgencia** | Media-alta (quieren iniciar trámite lo antes posible) |
| **Presupuesto** | $5,000 - $25,000 MXN para el proceso completo |

### Cliente Secundario: Consumidor

| Atributo | Descripción |
|---|---|
| **Situación** | Problema con banco, proveedor, servicio |
| **Canal** | Búsqueda Google "cómo demandar a [banco/empresa]" |
| **Urgencia** | Variable (algunos con plazos de PROFECO) |

### Cliente Secundario: Laboral

| Atributo | Descripción |
|---|---|
| **Situación** | Despido injustificado, falta de pago, sin finiquito |
| **Canal** | WhatsApp referido, búsqueda Google |
| **Urgencia** | Alta (plazo de 2 meses para demandar después de despido) |

---

## Anti-Patrones (NO es nuestro cliente)

- Despachos corporativos grandes (50+ abogados) — ya tienen sistemas
- Casos penales — fuera de scope del MVP
- Clientes que buscan asesoría legal gratuita sin intención de contratar
- Despachos sin presencia digital ni WhatsApp Business

---

## Métricas de Calificación

| Señal | Peso |
|---|---|
| Usa WhatsApp para coordinar con clientes | Alto |
| Tiene 5+ casos activos simultáneos | Alto |
| Se queja de documentos incompletos | Alto |
| Tiene cédula profesional verificable | Requisito |
| Está en zona urbana MX | Medio |
| Tiene página web o Google Business | Medio |
